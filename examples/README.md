# RoboPilot Examples

This directory contains small, static examples for RoboPilot's no-ROS-required workflows.

- `generated_projects/demo_detector/`: deterministic generated ProjectSpec demo package
- `ros1_migration_demo/`: ROS1-style package used by migration tutorials
- `ros2_scaffold_demo/`: scaffold-only ROS2 migration example for review
- `migration_outputs/`: sample migration plan and scaffold report artifacts
- `error_logs/`, `graphs/`, `pipelines/`, `prompts/`: small utility examples

Examples are for static inspection, documentation, and demonstrations. They do not require ROS, ROS2, `catkin_make`, `colcon`, launch execution, generated node execution, or robot hardware.
