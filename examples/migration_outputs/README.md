# Migration Output Samples

This directory contains small, deterministic sample artifacts for the no-ROS-required ROS1-to-ROS2 migration tutorial.

Included files:

- `migration_plan.yaml`: example YAML migration plan for `examples/ros1_migration_demo/`
- `migration_plan.json`: equivalent JSON-shaped sample for integration readers
- `scaffold_report.md`: example Markdown scaffold validation report

These files are examples only. They were not produced by running ROS, ROS2, `catkin_make`, `colcon`, launch files, or generated nodes.
