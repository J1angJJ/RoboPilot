# RoboPilot Migration Notes

Generated by RoboPilot as a conservative scaffold example.

This is not an automatic migration. RoboPilot did not run ROS, ROS2, `catkin_make`, `colcon`, launch files, generated nodes, or generated code.

Manual review is required before using this scaffold as a real ROS2 package.

Review items:

- Replace `rospy` code with reviewed `rclpy` code.
- Replace `roscpp` code with reviewed `rclcpp` code.
- Review ROS2 QoS choices for every publisher and subscriber.
- Review launch parameters, remaps, namespaces, and node names.
- Review message, service, and action interfaces under ROS2.
- Run real ROS2 build and runtime tests only in a ROS2 environment.
